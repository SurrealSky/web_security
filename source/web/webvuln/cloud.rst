云安全
================================

OSS对象存储
--------------------------------

bucket权限配置错误
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
+ Bucket授权策略中设置了ListObject

任意文件上传
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
+ 权限配置成了oss:PutObject，利用PUT上传
+ 一般脚本文件不会解析，没有执行权限
+ 尝试html文件是否解析
+ 篡改网站的图片，视频资源等

Bucket桶爆破
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
+ host前缀就是桶名，尝试爆破
+ bucket爆破成功之后，继续尝试爆破其中的key，就是文件

存储桶劫持
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
+ 当访问的域名出现NoSuchBucket关键字时，代表这个存储桶已被删除但是域名解析还没删除，我们可以创建一个名字一样的oss进行接管。
+ 注：腾讯云是无法接管的，阿里云可以。

aksk泄露
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
+ 搜索关键字如access_key、accesskey、accessKeySecret、secret。
+ 搜索渠道有GitHub、js代码、html前端、小程序反编译、app反编译、上传接口返回数据、heapdump泄露等，利用行云管家进行连接。

微信云安全
---------------------------------

应用场景
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
+ 微信公众号
+ 小程序
+ 企业微信

凭证泄露
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
+ 搜索关键字如appsecret 、 secret 、appid、corpid、corpsecret等。
+ 搜索渠道有GitHub、js代码、html前端、小程序反编译、app反编译、上传接口返回数据、heapdump泄露等，可以直接接管微信公众号或小程序。
